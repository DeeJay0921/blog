# XML
extensible markup language 可扩展标记语言。

## XML的作用
- 存储数据
- 作为配置文件
- 作为数据传输载体

## 定义XML
1. 文档声明： `<?xml version="1.0"?>`
> 指定encoding: `<?xml version="1.0" encoding="utf-8"?>`
还有standalone(yes/no)属性代表文档是否引用其他文件
2. 元素定义
- 元素即标签
- 必须要有一个根元素
- 可以写空标签，即自闭合标签
- 标签名可以完全自定义
3. 属性定义, 定义在元素内部，如`<stu id="111"></stu>`
